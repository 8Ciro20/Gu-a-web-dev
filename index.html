¿Como se escribe HTML?

Todo se basa en poner etiquetas, se abren "<" y se cierran con ">".

Tambien estan las etiquetas de cierre que sirven para terminar el comando de la etiqueta que se abrió posteriormente, se abren con "< / " y se cierran con ">"
(algunas etiquetas no necesitan etiqueta de cierre)

Empecemos por lo basico

Se empieza en el editor de codigo con la siguiente etiqueta " doctype! html ", esto para que el navegador con el que abrimos la pagina o documento sepa qué lenguaje tiene que interpretar.

Despues se usa la etiqueta html y se empieza a programar.

head Para la informacion que se le da al navegador de la pagina, dentro de esta etiqueta incluimos el titulos con la etiqueta title la cual es lo que se mostrara en la pestaña del navegador.

body Para escribir escribir toda la informacion de la pagina.

Tipos de letra con h seguido de un numero, usualmente son titulos porque resaltan.

strong Negrita hr es una linea y br es para cambiar de linea o renglón em Cursiva

p Para escribir un parrafo, este tomara un espacio al escribir esta etiqueta.

¿Como se ponen imagenes?

Para poner imagenes se usa la etiqueta img, la cual no hay que cerrar, para poner una url con imagen se usa el atributo src.

Para cambiar el tamaño de la imagen usamos los atributos width para el ancho y height para la altura seguido de igual y comillas dentro de las cuales pondremos el tamaño en pixeles


Si las imagenes no estan en linea solamente se escribe donde iría el link la direccion en la que se encuentra el archivo de la imagen en un dispositivo local.

¿Como se ordena el codigo según la estructura de la pagina?

Las siguientes etiquetas se utilizan para ordenar nuestro codigo por secciones de la pagina.

Estas etiquetas hacen lo mismo que un div, son contenedores de informacion pero se usan para organizar nuestro codigo.

  • header Se utiliza para la parte superior de la pagina.
  • Con nav especificamos links o botones que nos llevan a otras paginas del sito web.
  • Section se utiliza para indicar una seccion o tema en nuestra pagina.
  • footer se utiliza para indicar la parte inferior de la pagina.
  • Con la etiqueta main especificamos contenido particular de la pagina.

¿Como dar más accesibilidad a nuestra pagina?

Con el atributo alt dentro de la etiqueta img podemos dar detalles al narrador en pantalla para que describa la imagen cuando esta no se puede cargar.

Con el atributo lang dentro de la etiqueta html le decimos al navegardor en que lenguaje está la pagina, esto ayuda al lector en pantalla y a la visualizacion de algunos caracteres especiales de un idioma como la ñ en el español.

Con el atributo role en contenedores como div damos informacion al navegador de estos contenedores, si ponemos dentro de role banner esto le indicará al lector en pantalla que es el encabezado de la pagina.

Si ponemos dentro del atributo role esto ayudara al lector en pantalla a saber que hay imagenes dentro de ese contenedor.

Con el rol comment dentro del atributo role le indicamos al lector que el texto en el contenedor es un comentario.

Escribiendo el rol contentinfo indicamos al lector que es informacion del pie de la pagina, como derechos de autor, contacto, etc.

Usando el rol alert indicamos al lector de algun cambio o evento en la pagina por ejemplo "tu sesion expira en 10 segundos"

Para hacer que el lector no lea un parrafo se utiliza el atributo aria-hidden dentro de la etiqueta p .
